Это репозиторий для первой лабораторной работы.

Запуск программы: 
1. Открыть папку со скачанным jar файлом в консоли
2. Написать java -jar <название_jar_файла>.jar

Работа с программой
После запуска неолходимо написать в консоль ссылку на файл, который необходимо распарсить. 
1. Найдите нужный файл
2. Copy as path
3. Вставтье ссылку в терминал

Приложение выводит содержимое файла в заданном формате, а также сохраняет полученную информацию в отдельный файл в формате json
